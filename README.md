# ­ЪјИ Guitar Shop РђЊ Responsive Website

**Guitar Shop** рЃљрЃарЃўрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃњрЃўрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ **HTML**, **CSS** рЃЊрЃљ **JavaScript** рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ **responsive рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА** рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃљрЃќрЃћ рЃЎрЃЮрЃЏрЃцрЃЮрЃарЃбрЃБрЃџрЃљрЃЊ рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ.

рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃАрЃБрЃцрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА, рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃўрЃќрЃБрЃљрЃџрЃА рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋ рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљрЃА, рЃарЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃњрЃљрЃћрЃфрЃюрЃЮрЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃфрЃўрЃљрЃА рЃЊрЃљ рЃАрЃљрЃЎрЃЮрЃюрЃбрЃљрЃЦрЃбрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА.
